High Rise is a innovative take on the match 3 genre, for mobile devices and tablet.

I jumped on the project for a handful of months to develop an update and a new Android port. Highlights of my work include:

  • Implementing online leaderboards, both front and backend
  • Major refactor of the codebase to abstract a lot of the platform-specific code
  • Rework of IOS IAP and implementation of Android IAP through an abstraction layer
  • Creation of in-engine tooling by extending the Unity editor to improve debugging
  • Gameplay tweaks and bug fixes, mostly on saved game data, so called "undo" feature, UI rework, and so on
