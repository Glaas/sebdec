Seb Essentials

Seb Essentials is a collection of tools for Unity, created to avoid the hassle to re-create the same scripts over and over again. New features and tools will be added as I create more things that I want to re-use in the future.
Tools provided include :

  • FPS Player Controller or isit
  • 2D Player Controller
  • Item Animator (set-up bobbing and rotation)
  • GuiStyle presets
  • Mathematical curves library
  • Math functions and extensions

You can access the documentation in the wiki of this repository.
Contribution is strongly encouraged ! Please open issues and pull requests if you spot bugs or have some idea to improve or optimize.
